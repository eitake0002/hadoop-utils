# hadoop-utils
Hadoop utilities for hdfs and HiveQL.

- Create text data.
- Create Hive table.
- Load text data.
- Check HDFS metadata, HiveTable info etc.

supporing hadoop2.5.0 or later.

| Left align | Right align | 
|:-----------|------------:|
| OS       | CentOS6.0 or later| 
| Hadoop     | 2.5 or later     |
| Hive       | 0.14 or later    |
